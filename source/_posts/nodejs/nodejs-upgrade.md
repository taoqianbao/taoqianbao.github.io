## 前言

npm更新和nodejs更新

## 正文

更新你已经安装的NPM库，这个很简单，只需要运行。
```js
npm update -g
```
更新Nodejs自身。一直依赖我都是下载最新版的源码，然后make install，及其繁琐。其实只需要运行以下2个命令即可：

```js
npm install -g n 
n latest
```
